# Tarea 1 - Android

## Descripción
Aplicación Android con dos actividades.

## Funcionalidad
- **Primera actividad:** Muestra dos campos de texto para introducir nombre y edad, y un botón.
  - Si algún campo está vacío, muestra un Snackbar con un aviso.
  - Si ambos campos están rellenos, navega a la segunda actividad con los datos introducidos.
- **Segunda actividad:** Muestra en un TextView el nombre y la edad recibidos desde la primera actividad.

## Tecnologías
- Kotlin
- View Binding
- Intents
- Snackbar

## Requisitos
- Android Studio
- SDK mínimo: API 24
